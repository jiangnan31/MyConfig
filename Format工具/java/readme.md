#说明

Format工具：java formatter
依赖环境：eclipse/IDEA
安装部署：
1. Eclipse
Windows -> Preferences -> Java -> Code Style，Clean Up、Code Templates、Formatter分别选择Import，导入对应XML文件。(详见下载链接) 
如果不需要Copyright信息，想要自定义的，可以不导入Code Templates。
2. IDEA 
File -> Import Settings ->选择下载链接中的Baidu_IDEA.jar文件，可以看到两个选项，只需代码风格的，可以仅选择Code style schemes，如果需要默认的Copyright信息，选择Default Project settings。
